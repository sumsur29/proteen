# PROTEEEN — India Ka Protein Score 🧀💪

> Not a health app. A status signal. 💅

Interactive Protein Aura quiz that roasts you in Hinglish, generates a shareable card, and captures waitlist signups for the PROTEEEN app.

## Features
- 🧀 8 Protein Aura types (Paneer Mafia, Soya Sigma, Dal Delusional, etc.)
- 🟢🔴 Diet-aware questions (Veg / Non-Veg / Eggitarian)
- 🔥 Hinglish roasts with Bollywood references
- 📸 Canvas-generated shareable card (1080x1350 for Instagram stories)
- 📱 App mockup previews (Protein Score, Roast Mode, Wrapped, Squad Battles)
- 📧 Waitlist email capture

## Setup
```bash
npm install
npm run dev
```

## Deploy on Vercel
Connect this repo to Vercel — it auto-detects Vite. Zero config needed.

## Tech
- React 18 + Vite
- Canvas API for share card generation
- Zero external UI libraries — pure inline styles
